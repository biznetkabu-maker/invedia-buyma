## 💰 基準3：経済性チェック（重み 30%）

### A. 利益計算（必ず記入）

| 項目 | 計算式 | 金額 |
|------|--------|------|
| JPY仕入原価 | 現地価格 × 為替レート | ¥ |
| 関税 | JPY仕入原価 × 関税率（____%） | ¥ |
| 国際送料（固定） | | ¥ |
| BUYMA手数料 | BUYMA販売価格 × 手数料率（5.5〜7.7%） | ¥ |
| **総コスト** | 上記合計 | **¥** |
| **利益額** | BUYMA販売価格 − 総コスト | **¥** |
| **利益率** | 利益額 ÷ BUYMA販売価格 | **____%** |

> 🎯 **目標利益率: 15%以上**
> 10〜15%: 条件付き可 / 10%未満: 基本的に見送り / マイナス: 絶対に仕入れない

### B. 内外価格差

## 🛡️ 基準4：リスク管理チェック（重み 15%）

### A. 真正性

- [ ] 仕入れ先の種別（BUYMA規約に適合する仕入れ先か）:

  - [ ] ✅ 正規販売代理店・百貨店（例: SSENSE, Harrods, Saks, Harvey Nichols, Neiman Marcus）
  - [ ] ⚠️  BUYMA認定大手セレクトショップ（例: FARFETCH, MYTHERESA, NET-A-PORTER, MR PORTER, 24S）
  - [ ] ❌ 個人売買・出所不明 → **絶対に仕入れない**

- [ ] BUYMAの「バイヤー利用可能な仕入れ先リスト」に記載があるか確認済み

## 🔁 PurchaseEvaluator への入力（コード用）

```python
from purchase_evaluator import EvaluationInput, PurchaseEvaluator

inp = EvaluationInput(
    product_name="",          # 商品名
    brand="",                 # ブランド（例: "CELINE", "Saint Laurent"）
    model_year=2025,          # モデル年
    source_url="",            # 仕入れURL
    source_price=0.0,         # 現地価格
    currency="USD",           # 通貨
    exchange_rate=155.0,      # 為替レート
    buyma_price=0.0,          # BUYMA予定価格（JPY）
    japan_retail_price=0.0,   # 日本定価（JPY）
    dispatch_days=3,          # 現地発送までの日数
    japan_arrival_days=7,     # 日本着までの日数
    is_realtime_stock=True,   # リアルタイム在庫か
    packaging_quality="excellent",  # excellent / good / unknown
    buyma_rank=None,          # BUYMAランキング順位（不明はNone）
    sns_trending=False,       # SNSトレンド中か
    japan_soldout=False,      # 国内完売か
    japan_exclusive=False,    # 日本未入荷か
    favorites_count=0,        # お気に入り登録数
    has_cart_addition=False,  # カート投入あり
    source_type="authorized", # official / authorized / select / unknown
    is_volume_zone=True,      # 日本ボリュームゾーンか
    customs_rate=0.10,        # 関税率
    shipping_cost_jpy=2000,   # 国際送料（JPY）
    buyma_fee_rate=0.077,     # BUYMA手数料率（5.5〜7.7%）
    fx_buffer_rate=0.03,      # 為替バッファ率
    target_profit_rate=0.15,  # 目標利益率（10〜30%が目安、最低10%）
    product_category="",      # カテゴリ（例: "bag", "wallet", "sneaker"）
)

evaluator = PurchaseEvaluator()
score = evaluator.evaluate(inp)
print(score.summary())
```
